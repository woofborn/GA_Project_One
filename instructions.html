How to Play TypeQuest:

Story Mode:

  1. Read narrative (the part with a lot of words).
  2. Type the prompted word(s) into the input box.
  3. Type the words ACCURATELY.
  4. Try not to die.

Speed Mode:

  1. Select your preferred time limit in the dropdown timer.
  2. Type the prompted word(s) into the input box.
  3. Type the words ACCURATELY.
  4. Type as many words as you can in the allotted time.
